# PublisherComplexWeb

## Стек
* ASP.NET Core 8.0 - Web Api
* ASP.NET MVC 8.0 - Client
* Blazor UI - Client
* JWT token - Auth
* EF Core
* PostgreSql
* CQRS + MediatR
* Fluent Validation
* xUnit - test
* Docker
* gRPC
